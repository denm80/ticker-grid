# ticker-grid

## download the project
```
git clone https://github.com/denm80/ticker-grid.git
```

## install node

**nodejs v5.11.0 or higher is required**

## install npm modules
```
npm install
```

##install gulp
```
npm install gulp -g
```

## install .d.ts files
```
tsd update --save
```

## start server
```
gulp
```
